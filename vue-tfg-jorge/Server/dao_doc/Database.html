Class: Database

Database()

Clase para el acceso a base de datos (mysql)
Implemente primitivas de acceso a la BD:
  • connect(): Conectar a la BD.
  • close(): Cerrar la conexión.
  • query(): Ejecuta una sentencia SQL.
  • select(): Ejecuta una sentencia SELECT.
  • insert(): Ejecuta una sentencia INSERT.
  • update(): Ejecuta una sentencia UPDATE.
  • delete(): Ejecuta una sentencia DELETE.

Elimina una subscripción a un evento
Parameters:
Name Type Attributes Description
names string | array | number <optional>
 Nombre(s) de evento(s). Si es un número indica el id de la subscripción a eliminar.
filter Object <optional>
 Filtros de evento

Overrides:
  • Evented#off
Source:
  • evented.js, line 100
Example
obj.off(); // elimina todas las subscripciones
obj.off(7); // elimina la subscripcion con id 7 (valor retornado por 'on')
obj.off([2, 4]); // elimina la subscripción con ids 2 y 4
obj.off("myEvent"); // elimina todas las subscripciones a 'myEvent'
obj.off("myEvent", callback); // elimina todas las subscripciones a 'myEvent' con el callback indicado
obj.off("myEvent", { id: 7 }, callback); // elimina todas las subscripciónes de ese evento, con ese filtro y ese callback

select(tables, fields, where, order, options, callback)

Crea y ejecuta una sentencia SQL SELECT
Parameters:
Name Type Description
tables string | Array.<string> Tabla(s) para la SELECT
fields Object Campos a recuperar. Objeto donde la propiedad es el campo a retornar (AS) y el valor el campos de la BD
where Object Condiciones para el WHERE. Si el nombre empieza por ".", es un valor que no hay que escapar

Properties

Source:

Example
