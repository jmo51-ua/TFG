Class: RESTClient

RESTClient()

Cliente REST.
Permite el acceso a servicios REST.
Utiliza el formato de datos JSON.
Compatible con NodeJS y Angular.
Permite, en caso de desconexión temporar, almacenar y reenviar peticiones. Se controla mediante la propiedad resend.
En caso de necesiar login previo usar el método login(). Métodos básicos (compatibles con callback y Promise):
  • get(): Realiza una petición GET.
  • post(): Realiza una petición POST.
  • put(): Realiza una petición PUT.
  • delete(): Realiza una petición DELETE.
