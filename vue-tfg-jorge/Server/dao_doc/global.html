EntitySchema

Esquema de una entidad. Atributos que definen como es una entidad para automatizar la creación del DAO y de los servicios REST.
Type:
  • Object
Properties:
Name Type Attributes Default Description
name string Nombre del esquema. Por este nombre se referencia a la entidad en el dao. Si no se indica y se carga el esquema de un archivo, se toma el nombre del archivo como nombre del esquema
title string Título del esquema
description string Descripción del esquema
table string | Array.<string> <optional>
 Tabla(s) a las que está asociada la entidad
limit integer <optional>
 0 Establece un limite de registros en las SELECT
distinct boolean <optional>
 false Recupera de la SELECT sólo registros distintos 'SELECT DISTINCT'. 0 indica todos
urlItem string <optional>
 URL para el acceso REST a un elemento (item)
urlList string <optional>
 URL para el acceso REST a una lista (list)
put boolean <optional>
 Indica si se puede hacer PUT sobre la entidad. Por defecto true si hay urlItem
post boolean <optional>
 Indica si se puede hacer POST sobre la entidad. Por defecto true si hay urlList
delete boolean <optional>
 Indica si se puede hacer DELETE sobre la entidad. Por defecto toma el valor de put
extends string <optional>
 Extiende la entidad de otra (herencia). Al extender se copian las propiedades y listas de la clase extendida
properties Object <optional>
 Objeto con las propiedades de la entidad. El valor de cada propiedad es de tipo module:entitySchema~EntitySchemaProperty
lists Object <optional>
 Objeto con las listas de la entidad (sub entidades). El valor de cada lista es de tipo module:entitySchema~EntitySchemaList
