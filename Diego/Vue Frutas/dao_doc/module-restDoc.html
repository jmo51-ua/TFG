Methods

(static) restDoc(config, dao, authopt)

Crea un servidor REST (basado en express) a partir de un DAO.
Por cada entidad del DAO crea:
  • GET urlList: Lista elementos de la entidad
  • GET urlItem: Obtiene un elemento de la entidad
  • SET urlItem: Modifica un elemento de la entidad
  • POST ulrList: Crea un nuevo elemento de la entidad
  • DELETE urlItem: Elimina un elemento de la entidad
Parameters:
Name Type Attributes Description
config module:restServer~Config Configuración del servidor REST
dao DAOServer Referencia al DAO
auth function <optional>
 Función de autorización: auth(credentials, (userId) => {...})
